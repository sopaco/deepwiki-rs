# 输出模块 (generator/outlet)

## 这个模块在做什么

输出模块是 Litho 流水线的"质检与出厂"环节——它把编排阶段组装好的 DocTree 文档树持久化到磁盘，修复 Mermaid 图表语法，生成执行摘要报告。这个模块看似简单（只是"写文件"），但 Mermaid 修复和摘要生成两个功能实际上很关键：前者保证了文档的可读性（Mermaid 图语法错误会让架构图完全无法渲染），后者提供了执行过程的透明度（让用户知道哪些步骤成功了、哪些被跳过、整体耗时如何）。

## 核心功能点

1. **文档持久化**——`DiskOutlet::save()` 把 DocTree 中的所有文档内容写入用户指定的输出目录，每个文档章节对应一个独立的 Markdown 文件。
2. **Mermaid 图表修复**——`MermaidFixer` 调用外部 mermaid-fixer 二进制工具，验证和修复文档中所有 Mermaid 图表的语法错误。
3. **执行摘要生成**——`SummaryOutlet::save()` 通过 `SummaryContentGenerator` 和 `SummaryDataCollector` 生成一份执行摘要报告，包含各阶段的耗时、Agent 执行状态、缓存命中率等信息。

| 组件 | 文件路径 | 一句话职责 |

| `SummaryContentGenerator` | `src/generator/outlet/summary_generator.rs` | 通过LLM生成摘要内容 |
| `DocTree` | `src/generator/outlet/mod.rs` | 文档内容的树形容器 |
